Fassen Sie die Kapitel Discover, Define, Design, Deliver, Distribute in jeweils einer Seite zusammen. Überführen Sie die Ergebnisse in eine Website, die Sie entweder selbst in HTML/CSS erstellen, bzw. einen geeigneten Websiten­konfigurator verwenden (Webflow, WIX, Squarespace).

Die Website besteht aus 6 Seiten, Startseite, Kapitel, die 3 Sprints und das Buch. Es steht auch ein Link zu der Impressum-Seite im Footer, welcher jedoch nur als Platzhalter dient. Durch den "Kapitel"-Tab erreicht man die Kapitel-Seite. Hier befinden sich weitere, im Viewport fixierte Tabs mit den Kapitelnamen, die beim Anklicken jeweils den dazugehörigen Text offenbaren. Dies habe ich getan, um bei der eigentlichen Navigation-Bar oben Platz zu sparen. Zudem erreicht man durch die "Sprint"-Tabs die Sprint-Seiten, die zur Sammlung und Darstellung meiner Ergebnisse aus den einzelnen Sprints im Kurs dient. Zuletzt gibt es auch noch eine Buch-Seite, auf der das Buch "Branded Interaction" und die Autoren, Marco Spies und Katja Wenger, nochmal vorgestellt werden, da ich versucht habe die Website so aufzubauen, dass nicht-Leser sich zurecht finden und den Inhalt verstehen würden.

Diese Website wurde bzw. wird mit HTML, CSS und JavaScript gebaut. Ich habe den Code in Visual Studio Code geschrieben und auf GitHub hochgeladen, hier kann man sich den Code anschauen. Die Website wird dementsprechend auch auf GitHub Pages gehostet.
Der Grund, warum ich mich dazu entschieden habe den Code selbst zu schreiben, ist einfach nur der, dass ich versuche mir das Programmieren selbst beizubringen und diese Website als eine gute Übung diente. Dies ist auch die erste "vollständige" Website, die ich geschrieben haben.

Schwachstellen

- Die Website ist nicht responsive, kein Mobile-First Approach
- Je nach Browser können Ungenauigkeiten bei der Darstellung entstehen (z.B. Abstände)
- Bilder brauchen manchmal einige Sekunden um zu laden

Weitere Schritte?

- Momentan gestalterisch schlicht, kann weiter ausgebaut werden
- Website kann um weitere Tabs/Inhalte erweitert werden, wie z.B. Glossar, Methoden, usw.
